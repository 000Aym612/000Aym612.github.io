My Journey with JavaScript

I've noticed that JavaScript shares similarities with both Python and Java/C/C++. Its approach to functions, where the data type of parameters or the return type isn't explicitly stated, is reminiscent of Python's flexibility. This is a contrast to the strict type definitions required in Java, C, or C++.

Unlike Java or C, JavaScript doesn't require a main function to execute code. However, its grammar, including elements like switch statements and increment/decrement operators, aligns more closely with Java and C.

Exploring JavaScript's Utility in Software Engineering

As a newcomer to JavaScript, I'm still exploring its utility in software engineering. The Workout of the Day (WOD) exercises in my class have been instrumental in applying basic JavaScript concepts to practical scenarios, enhancing my understanding and guiding me in the right direction.

Concerns and Challenges

One challenge I face is integrating JavaScript with HTML/CSS. Although my background in data science and data analysis involves different skill sets, I'm concerned about my unfamiliarity with HTML/CSS, which seems essential for effective JavaScript usage.
